Getting Started¶

Creating an experiment¶

  1. Create a new Experiment in the Django admin.
  2. Name your experiment in the Title field.
  3. Set ‘Multivariate’ as the Experiment type.
  4. Put in the URL of the representative page for the Test page URL. For example, if you are testing a blog entry page, pick a recent blog entry for the page url.
  5. Put in the same URL for the Conversion page URL
  6. Leave Auto prune mode at None, unless you really know what you’re doing.
  7. Save the experiment.

Create a section for an experiment¶

  1. Each listing of experiments includes a field detailing how many sections are currently defined for that experiment and links to view the experiment’s sections or add a section. Click on Add section.
  2. The Experiment field will already be set, so simply enter in the Title for this section.
  3. Clicking on Save will save this section and take you to the list of all sections. Clicking on Save and add another will save this section and allow you to add a new section.

Create a variation for an experiment¶

  1. Each list of sections includes a field detailing how many variations are currently defined for that section and links to view the section’s variations or add a variation to the section. Click on Add variation.

Tagging the template¶

Before you can go on with Website Optimizer, you need to tag the pages so google can validate them. Django-GWO provides some easy tags to make the process easier than the instructions in Website Optimizer.

  1. At the top of the template, add {% load gwo_tags %}.

  2. In the <head> section of the page add {% gwo_control_script ExperimentName %}. This inserts the copied control script for ExperimentName into the page.

  3. Just before the </body> tag of the template add {% gwo_tracking_script %}

  4. Around each section that you are testing add:

    {% gwo_section SectionName %}

... your original content ...

{% end_gwo_section %}

    The SectionName is passed through directly to the Website Optimizer JavaScript for the section name.
